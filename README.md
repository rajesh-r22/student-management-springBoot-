# 🎓 Student Management API

A Spring Boot REST API for managing student records, built with PostgreSQL and documented using Swagger/OpenAPI.  
This project is part of my **50 Java Backend Projects Challenge** to sharpen skills and showcase consistent progress.

---

## 🚀 Features
- ➕ Add new students
- 🔍 Fetch student by ID
- 📋 Fetch all students
- ✏️ Update student details
- ❌ Delete student records
- 📖 Interactive API documentation via Swagger UI

---

## 🛠️ Tech Stack
- **Java 17**
- **Spring Boot 3.2.5**
- **Spring Data JPA**
- **PostgreSQL**
- **Lombok**
- **Swagger / Springdoc OpenAPI**

---

## 📖 API Documentation
Once the application is running:
- Swagger UI → [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
- OpenAPI Spec → [http://localhost:8080/v3/api-docs](http://localhost:8080/v3/api-docs)
